JavaScript Data Types

Introduction

JavaScript supports different data types, which are divided into primitive and non-primitive (also called reference) types. These types are used to represent different kinds of values in the language.

Primitive Data Types

Primitive types are basic data types that are immutable and are not objects. These include:

  • Number: Represents both integer and floating point numbers.
  • String: Represents a sequence of characters enclosed in quotes.
  • Boolean: Represents a logical entity that can be either true or false.
  • Undefined: A variable that has not been assigned a value is of type undefined.
  • Null: Represents the intentional absence of any object value.
  • Symbol: A unique and immutable value, typically used as identifiers for object properties.
  • BigInt: Represents large integers that can't be represented by the Number type.

Non-Primitive Data Types

Non-primitive types are reference types, meaning they are objects and can store collections of data. The most common non-primitive types are:

  • Object: A collection of key-value pairs, such as arrays, functions, or other complex structures.
  • Array: An ordered collection of elements, where each element is accessible by an index.
  • Function: A block of code designed to perform a particular task, often returning a value.

Examples


// Example of Primitive Types
let num = 42;            // Number
let name = "Anurag";     // String
let isActive = true;     // Boolean
let unknownValue = null; // Null
let uniqueSymbol = Symbol('symbol'); // Symbol
let bigNumber = 1234567890123456789012345678901234567890n; // BigInt

// Example of Non-Primitive Types
let person = { name: "Anurag", age: 21 }; // Object
let numbers = [1, 2, 3, 4, 5];            // Array
function greet() { console.log("Hello!"); } // Function

Type Conversion

JavaScript also allows you to convert between data types, such as from string to number, or boolean to string:

  • String to Number: Number("42")
  • Number to String: String(42)
  • Boolean to String: String(true)

Conclusion

JavaScript data types are fundamental to how the language works. Understanding how to use and manipulate them is crucial for effective programming in JavaScript.
