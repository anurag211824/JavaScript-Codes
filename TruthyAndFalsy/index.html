Truthy and Falsy Values in JavaScript

Falsy Values (Only 8 in JavaScript)

These values evaluate to false in a boolean context:

  • false
  • 0 (Number zero)
  • -0 (Negative zero)
  • 0n (BigInt zero)
  • "" (Empty string)
  • null
  • undefined
  • NaN (Not a Number)

Truthy Values

Everything except the above falsy values is considered truthy in JavaScript. Here are some common examples:

  • true
  • Any non-zero number (e.g., 5, -3, 3.14)
  • Any non-empty string (e.g., "hello", '0', "false")
  • Any object (e.g., {}, [])
  • Any array (even empty [])
  • Any function (e.g., function() {})
  • Any non-zero BigInt (e.g., 1n, -10n)
  • Infinity and -Infinity
